
# lexical
Задача
------

Написать скрипт, который выводит все строки, лексикографически и по длине,
находящиеся между двумя заданными, и состоящие только из строчных латинских букв.

Оцениваться будет не только алгоритмическая верность, но и архитетура всего решения.

`./strings --from a --to bb`
